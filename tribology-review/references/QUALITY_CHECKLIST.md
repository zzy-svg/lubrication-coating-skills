### Hedging Language Examples | 谨慎表达语言示例

**Appropriate | 适当**:
```markdown
"...suggests that DLC coatings may provide..."
"...appears to indicate improved wear resistance..."
"...has demonstrated potential for..."
"...may be attributed to the high sp³ content..."

**Avoid | 避免**:
```markdown
"...proves that DLC is the best coating..."
"...always shows superior performance..."
"...demonstrates conclusively..."
```

## Common Issues and Fixes | 常见问题与解决方案

| Issue | Solution |
|-------|----------|
| Too few references | Expand literature search to more databases |
| Missing quantitative data | Re-read key papers for numerical values |
| Inconsistent terminology | Use CLAUDE.md glossary for consistency |
| Weak conclusions | Ensure conclusions match evidence presented |
| Poor figure captions | Add detailed descriptions of all figure elements |
| Outdated references | Prioritize papers from 2019-2024 |
| Overly general statements | Add specific data points and citations |
| Missing limitations section | Add honest assessment of coating drawbacks |
