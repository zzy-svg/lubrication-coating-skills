## 4. Literature Collection Workflow | 文献收集工作流

### Phase 1: Broad Search | 阶段1：广泛搜索

1. **Initial Query**: Use broad keywords to identify major papers
   ```
   "tribology coating review"
   "solid lubricant coating" review
   "DLC coating" comprehensive review
   ```

2. **Date Filter**: Focus on recent literature (2019-2024)
3. **Source Identification**: Note journals with most relevant papers

### Phase 2: Targeted Search | 阶段2：定向搜索

1. **Specific Topics**: Search for each coating type
   ```
   "MoS2 coating" tribology
   "DLC coating" high temperature
   "self-lubricating" composite
   ```

2. **Method Search**: Search for deposition methods
   ```
   "PVD DLC coating" review
   "sputtering MoS2" optimization
   ```

3. **Application Search**: Search by application area
   ```
   "DLC coating" aerospace
   "tribological coating" automotive
   ```

### Phase 3: Citation Tracking | 阶段3：引文追踪

1. **Backward**: Check references of key papers
2. **Forward**: Use Google Scholar "Cited by" feature
3. **Related**: Find related papers through "Related articles"

### Phase 4: Quality Filtering | 阶段4：质量筛选

| Criteria | Importance | Check |
|----------|------------|-------|
| Peer-reviewed | Essential | Journal impact factor |
| Recent publication | High | 2019-2024 preferred |
| Citation count | Medium | Highly cited = influential |
| Reproducible methods | High | Detailed methodology |
| Quantitative data | Essential | COF, wear rate reported |

## 6. Recommended Search Workflow | 推荐搜索工作流

### Quick Start | 快速开始

```bash
# Day 1-2: Broad Overview
1. Search: "tribology coating review" (Google Scholar, 50 papers)
2. Identify: Top 10 most relevant reviews
3. Read: Abstracts, identify subtopics

# Day 3-4: Focused Search
4. Search: Each subtopic with specific keywords
5. Collect: 20-30 papers per subtopic
6. Categorize: By coating type, application, method

# Day 5-6: Deep Dive
7. Search: Recent advances (2019-2024)
8. Read: Full text of key papers
9. Extract: Data for comparison tables

# Day 7: Synthesis
10. Organize: Literature matrix
11. Identify: Research gaps
12. Plan: Review structure
```

### Quality Papers Identification | 优质论文识别

| Indicator | Good Sign | Bad Sign |
|-----------|-----------|----------|
| Journal | Q1/Q2 | Conference only |
| Citations | >50 | <5 (recent) |
| Methodology | Detailed | Vague |
| Data | Quantitative | Qualitative only |
| Figures/Tables | Clear | Poor quality |
