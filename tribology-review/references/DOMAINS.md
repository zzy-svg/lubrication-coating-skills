# Coating Domains and Categories | 涂层领域与分类

This document provides comprehensive coverage of lubrication coating categories, their properties, applications, and comparative analysis for literature review writing.

本文档为文献综述撰写提供润滑涂层分类、特性、应用和对比分析的全面覆盖。

---

## 1. Solid Lubricant Coatings | 固体润滑涂层

### 1.1 Layered Materials | 层状材料

#### MoS₂ (Molybdenum Disulfide) | 二硫化钼

| Property | Value | Unit | 说明 |
|----------|-------|------|------|
| Crystal Structure | 2H-MoS₂ | - | 六方晶系 |
| Temperature Range | RT - 350°C | °C | 工作温度范围 |
| COF (Vacuum) | 0.01 - 0.03 | - | 真空环境 |
| COF (Air) | 0.03 - 0.20 | - | 空气环境 |
| Wear Rate | 10^-6 - 10^-5 | mm³/Nm | 磨损率 |
| Hardness | 1 - 10 | GPa | 硬度 |
| Layer Spacing | 0.62 | nm | 层间距 |

**Advantages | 优势**:
- Extremely low COF in vacuum and dry environments
- Excellent load-carrying capacity
- Good chemical stability

**Limitations | 局限性**:
- Oxidizes above 350°C in air
- Humidity-sensitive performance
- Poor adhesion to some substrates

**Doping Strategies | 掺杂策略**:
| Dopant | Effect | Application |
|--------|--------|--------------|
| Ti | Improved hardness | High-load applications |
| Au | Enhanced oxidation resistance | High-temperature |
| Sb₂O₃ | Extended wear life | Long-duration sliding |
| C | Improved toughness | Composite coatings |

#### WS₂ (Tungsten Disulfide) | 二硫化钨

| Property | Value | Unit | 说明 |
|----------|-------|------|------|
| Crystal Structure | 2H-WS₂ | - | 六方晶系 |
| Temperature Range | RT - 500°C | °C | 比MoS₂更高 |
| COF | 0.02 - 0.05 | - | 真空/干燥环境 |
| Wear Rate | 10^-6 - 10^-5 | mm³/Nm | 与MoS₂相当 |
| Hardness | 2 - 15 | GPa | 略高于MoS₂ |

**Advantages over MoS₂ | 相比MoS₂的优势**:
- Better oxidation resistance (up to 500°C)
- More stable in humid environments
- Higher thermal stability

#### h-BN (Hexagonal Boron Nitride) | 六方氮化硼

| Property | Value | Unit | 说明 |
|----------|-------|------|------|
| Structure | Hexagonal | - | 类石墨结构 |
| Temperature Range | RT - 900°C | °C | 超高温稳定性 |
| COF | 0.15 - 0.40 | - | 空气环境 |
| Hardness | 1 - 30 | GPa | 可调控 |
| Thermal Conductivity | 200 - 400 | W/mK | 高导热 |

**Applications | 应用**:
- High-temperature furnace components
- Hot-pressing dies
- Aerospace applications

#### Graphene and Graphite | 石墨烯与石墨

| Property | Graphene | Graphite | Unit |
|----------|----------|----------|------|
| COF (Dry) | 0.01 - 0.10 | 0.15 - 0.30 | - |
| Temperature Range | RT - 600°C | RT - 500°C | °C |
| Thickness | Single layer | Bulk | - |
| Hardness | ~130 | 0.5 - 1 | GPa |

**Advantages | 优势**:
- Ultra-low friction at nanoscale
- High strength-to-weight ratio
- Excellent thermal stability

### 1.2 Soft Metal Lubricants | 软金属润滑剂

## 5. Multilayer/Composite Coatings | 多层/复合涂层

### 5.1 Design Strategies | 设计策略

| Strategy | Description | Benefits |
|----------|-------------|----------|
| Hard/Soft Alternation | TiN/Ti, CrN/Cr | Crack deflection |
| Gradient Structure | Composition gradient | Stress relaxation |
| Nanolayer | < 100 nm layers | Superlattice effect |
| Superlattice | Alternating nm layers | Enhanced hardness |

### 5.2 Common Multilayer Systems | 常见多层体系

| System | Layer Thickness | Hardness | COF | Applications |
|--------|-----------------|----------|-----|--------------|
| TiN/Ti | 2-5 nm / 2-5 nm | 30-40 GPa | 0.4-0.6 | Cutting tools |
| CrN/Cr | 3-7 nm / 3-7 nm | 25-35 GPa | 0.3-0.5 | Forming tools |
| DLC/MoS₂ | 1-3 μm / 0.5-2 μm | 15-25 GPa | 0.05-0.15 | Space applications |
| TiAlN/CrN | 3-5 nm / 3-5 nm | 35-45 GPa | 0.3-0.5 | High-speed cutting |

### 5.3 Advantages | 优势

- **Crack deflection**: Interlayers arrest crack propagation
- **Stress relaxation**: Gradient reduces thermal stress
- **Tailored properties**: Combine hard/soft characteristics
- **Improved adhesion**: Bonding layers enhance substrate attachment

## 7. Application Guidelines | 应用指南

### 7.1 by Industry | 按行业分类

| Industry | Recommended Coatings | Key Requirements |
|----------|---------------------|------------------|
| Aerospace | DLC, MoS₂, TiAlN | Low weight, reliability |
| Automotive | DLC, CrN, TiCN | High volume, cost-effective |
| Cutting Tools | TiAlN, AlCrN, DLC | High speed, long life |
| Biomedical | DLC, TiN | Biocompatibility |
| Marine | CrN, DLC | Corrosion resistance |
| Space | MoS₂, WS₂, DLC | Vacuum compatibility |

### 7.2 by Environment | 按环境分类

| Environment | Recommended Coatings | COF Target | Special Considerations |
|-------------|---------------------|------------|----------------------|
| Vacuum | MoS₂, WS₂, DLC | < 0.1 | No outgassing |
| High Temperature | TiAlN, AlCrN, h-BN | 0.3-0.5 | Oxidation resistance |
| Humid | DLC, TiN, CrN | 0.1-0.3 | Hydrolysis resistance |
| Corrosive | CrN, DLC (doped) | 0.1-0.3 | Chemical stability |
| Cryogenic | DLC, MoS₂ | 0.05-0.2 | Thermal shock |
